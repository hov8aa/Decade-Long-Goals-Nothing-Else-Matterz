# 10MinutesGoals

a simple app for focussed work

Pick one thing. Do it for 10 minutes. Nothing else matters.

## What it does

- **Today** — fresh start every day. Type a goal (or pick up a past one) and hit Start.
- **Timer** — a full-screen 10-minute countdown. Finish it or give up; both get recorded.
- **History** — every session, day by day. Tap any past goal to run it again.

Single player, no login, no backend — everything lives on your device.

## Run it

Built with [Expo](https://expo.dev) (SDK 56) — one codebase for iOS, Android, and web.

```bash
npm install
npm run web        # in the browser
npm start          # scan the QR with Expo Go for phone
```
