# 10MinutesGoals

a simple app for focussed work

The core is **one decade-long goal** — the north star. The 10-minute session is the bare-minimum unit of action that keeps it alive today. Commit to the decade, do 10 minutes toward it, nothing else matters.

The rule of tens: **10 seconds to commit · 10 taps to evolve · 10 minutes to act · 10 years to arrive.**

(How the project got here — including its own gaps — is in [EVOLUTION.md](EVOLUTION.md). What this version promises and refuses to promise is in [REQUIREMENTS.md](REQUIREMENTS.md).)

## What it does

- **The ceremony** — before anything else exists: write the goal and its reasoning, read the terms, retype it by hand, hold for ten seconds. Then the decade begins.
- **Today** — fresh start every day, the north star as a permanent header. Type the one thing (or pick up a past one), hit Start.
- **Timer** — a full-screen 10-minute countdown showing what it serves. Finish it or give up; both get recorded.
- **The north star** — one tap to see the version trail; ten taps to wake evolution, which demands its reasoning. It never resets, only evolves.
- **History** — one river: sessions and evolutions interleaved, each session pinned to the exact wording it served.

Single player, no login, no backend — everything lives on your device.

## Run it

Built with [Expo](https://expo.dev) (SDK 56) — one codebase for iOS, Android, and web.

```bash
npm install
npm run web        # in the browser
npm start          # scan the QR with Expo Go for phone
```
